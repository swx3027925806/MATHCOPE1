##  天下苦公式久矣

作为一个经常要和公式打交道的孩子，我被公式编辑虐的体无完肤！记得18年那会，有mathpix的救赎，艰难度日，然19年就开始收费了。虽然它也有API吧，但我没有信用卡······

一年来，我饱受煎熬，于是终于下定决心去嫖个API来完善一个公式识别的工具，于是这个mathcopy就出炉了。【一天查找API，一个下午写】非常简陋，不要嫌弃~

##  使用方法

1. 首先截图，或者从网站上保存图片，保证图片在剪切板里
2. 运行mathcopy.exe
3. API识别完成后会将letax文本替换到剪切板中，可以直接粘贴。
